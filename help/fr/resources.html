Contacter les développeurs et les autres utilisateurs

Si vous souhaitez poser des questions concernant l'utilisation de Fontmatrix, vous pouvez soit rejoindre la liste de diffusion soit notre canal IRC sur le réseau Freenode (nous avons intégré un client de discussion web sous Fast IRC que vous pouvez utiliser si vous n'êtes vraiment pas habitué à cette chose qu'est l'IRC).

Vous pouvez également rejoindre un forum sur le site web Libre Graphics World et y poser vos questions.

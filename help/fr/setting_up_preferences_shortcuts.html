Raccourcis

Ici, vous pouvez configurer les raccourcis (ou raccourcis clavier — quel que soit le mot que vous trouvez le plus approprié pour les combinaisons de touches qui permettent d'accéder aux éléments de menu sans avoir à passer par les menus en premier lieu).

Le tableau affiche les actions, leurs raccourcis (le cas échéant) et des conseils pour les actions au cas où vous ne seriez pas sûr de ce que fait une action particulière.

Pour modifier un raccourci existant pour un élément de menu ou en ajouter un manquant :

  1. Choisissez une action dans la liste
  2. Cliquez sur le bouton "Changer"
  3. Appuyez sur la combinaison de touches

La nouvelle combinaison sera affichée dans le tableau des raccourcis et à droite du bouton "Supprimer".

Pour supprimer un raccourci existant pour un élément de menu :

  1. Choisissez une action dans la liste
  2. Cliquez sur le bouton "Supprimer"

Le raccourci sera supprimé.

Veuillez noter que toutes les modifications apportées aux raccourcis ne peuvent pas être annulées.

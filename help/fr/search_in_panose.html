Recherche par métadonnées PANOSE

L'implémentation actuelle de la recherche par métadonnées PANOSE est plutôt expérimentale. Nous voulions juste avoir quelque chose avec quoi jouer, que cela ait l'air bien ou non, nous continuerons donc à y travailler (et écrirons certainement une meilleure documentation sur l'utilisation de PANOSE dans Fontmatrix). En attendant, voici comment utiliser ce que vous avez jusqu'à présent.

Dans l'onglet Classification, vous pouvez voir un tableau qui affiche une sorte de carte PANOSE pour les polices actuellement disponibles. Cette carte se compose de plusieurs axes, chacun d'eux représentant une variable PANOSE particulière. Si vous choisissez une police dans la barre latérale, vous verrez une fine ligne noire allant d'un axe à l'autre.

Pour trouver des polices correspondant à certaines valeurs de variables PANOSE, passez d'abord votre souris sur l'axe/la variable. Cliquez ensuite sur une valeur et regardez la barre latérale pour voir que seules les polices qui correspondent à cette valeur sont filtrées. Vous pouvez appuyer sur Maj et cliquer sur une autre valeur du même axe/variable pour étendre la recherche. Si le nombre de polices filtrées est trop important, vous pouvez affiner davantage la recherche en passant votre souris sur un autre axe et en choisissant une valeur d'intérêt. Notez que Fontmatrix construira une requête pour vous dans la partie supérieure de la barre latérale Parcourir les polices.

Parfois, les informations PANOSE fournies par les fournisseurs de polices sont incorrectes, ce qui entraîne des résultats incorrects pour vos requêtes. Vous pouvez résoudre ce problème en modifiant les métadonnées PANOSE pour une police particulière sans enregistrer les modifications dans un fichier de police réel. Apprenez comment faire.

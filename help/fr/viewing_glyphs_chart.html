Tableau des glyphes

En mode Glyphes, Fontmatrix peut afficher un tableau des glyphes présents dans la police actuellement sélectionnée. Il regroupe également les glyphes en blocs Unicode et affiche la couverture de ce bloc en %.

En cliquant sur un glyphe dans le tableau, vous afficherez sa version agrandie avec ses métriques et sa catégorie.

Si vous n'êtes pas sûr du bloc qui contient le glyphe d'intérêt, passez au regroupement Afficher tous les glyphes mappés et dans le champ de recherche à droite, tapez le début du nom du glyphe. Fontmatrix complétera automatiquement le nom et si le glyphe est présent, il fera défiler le tableau et fera clignoter ce glyphe un instant.

Cliquer sur le bouton Mode Copie active le champ de saisie que vous pouvez utiliser comme une casse numérique. Activez-le simplement et commencez à cliquer sur les glyphes pour les copier dans ce champ. Ensuite, sélectionnez tous les caractères copiés et copiez-les dans le presse-papiers pour les coller ailleurs.

Configuration

Fontmatrix est une application très configurable. La plupart des options que vous préférerez configurer une fois pour toutes sont accessibles via l'élément de menu Éditer > Préférences. Les options du moteur de mise en page de texte sont de nature interactive et sont donc disponibles dans une boîte de dialogue séparée (Service > Options du moteur de mise en page de texte) qui reste toujours au-dessus.

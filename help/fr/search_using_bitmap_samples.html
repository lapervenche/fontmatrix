Recherche par échantillons bitmap

Si vous possédez une énorme collection de polices, vous pouvez rechercher une police en utilisant un échantillon bitmap de l'un des glyphes. Il existe deux façons de le faire : vous pouvez soit fournir à Fontmatrix un fichier bitmap prêt à l'emploi, soit utiliser le mode Capture qui récupérera une image bitmap directement depuis votre écran.

Lorsque vous avez l'image dans la prévisualisation de l'échantillon raster, cliquez sur le glyphe pour que Fontmatrix calcule sa boîte englobante. Si Fontmatrix s'est trompé, activez la case à cocher Ajuster le cadre et faites glisser les coins d'un cadre autour de la boîte englobante présumée jusqu'à ce qu'elle soit correcte.

Maintenant, tapez le caractère qui se trouve sur l'échantillon et appuyez sur le bouton Rechercher.

Vous pouvez également regarder une vidéo de démonstration de cette fonctionnalité.

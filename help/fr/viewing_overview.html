Aperçu de la police

Les informations de base sur la police sont résumées dans l'onglet Informations sur la police, qui est ouvert par défaut lorsque vous démarrez l'application. Il liste :

  • le nom complet de la police et son emplacement dans le système ;
  • le nombre de glyphes et le type du fichier de police ;
  • des informations sur le concepteur et le fournisseur ;
  • l'URL de la licence et/ou le texte intégral de la licence, le cas échéant ;
  • les métadonnées PANOSE, le cas échéant ;
  • les conditions d'utilisation (intégration, édition, etc.).

Étant donné que Fontmatrix utilise Webkit pour rendre ces informations, son apparence est quelque peu personnalisable via CSS. Vous pouvez créer un fichier CSS personnalisé, l'enregistrer dans /usr/share/fontmatrix/resources/ et le choisir dans Éditer > Préférences > Affichage.

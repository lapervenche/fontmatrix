Création de catalogues de polices

Les catalogues de polices sont probablement le moyen le plus simple de présenter les polices que vous avez conçues à vos clients potentiels. Fontmatrix facilite grandement la création de catalogues de polices.

Par exemple, vous devez créer un catalogue de toutes les polices conçues par votre entreprise en 2009.

  1. Allez dans la barre latérale Parcourir les polices.
  2. Dans le menu déroulant des étiquettes, choisissez "Fabricant", tapez le nom de votre entreprise et appuyez sur Entrée.
  3. Activez la case à cocher Et.
  4. Dans le menu déroulant des étiquettes, choisissez "Copyright", tapez "2009" et appuyez sur Entrée.

Cela affinera la liste des polices disponibles pour n'afficher que les polices qui ont été créées ou mises à jour en 2009 par votre entreprise. Vous voudrez maintenant probablement accéder à cette boîte de dialogue :

  1. Allez dans Fichier > Exporter le catalogue de polices....
  2. Dans la boîte de dialogue nouvellement ouverte, choisissez un modèle.
  3. Cliquez sur Parcourir... pour définir le nom du fichier PDF de sortie.
  4. Cliquez sur OK pour créer le catalogue de polices.

Service

Cet onglet regroupe les options liées aux outils et ressources externes.

L'Éditeur de polices vous permet de définir l'éditeur de polices préféré et utilise par défaut FontForge, qui est utilisé sur les systèmes Linux et Mac (veuillez noter que les principaux développeurs de Fontmatrix sont des utilisateurs GNU/Linux).

L'option Navigateur d'aide vous permet de définir le navigateur à utiliser pour lire ce système d'aide. Par défaut, il utilise le propre navigateur d'aide de Fontmatrix qui utilise Webkit pour le rendu des pages d'aide.

L'option Dictionnaire de césure vous permet de définir un dictionnaire à utiliser avec le texte d'exemple. Les options Gauche et Droite définissent FIXME.
